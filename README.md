Vimfiles
========

 ## Install

 ``` bash
 $ git clone https://github.com/jaimelopesflores/vimfiles.git ~/.vim
 $ echo "source ~/.vim/vimrc" > ~/.vimrc
 $ cd ~/.vim
 $ git clone https://github.com/VundleVim/Vundle.vim.git ~/.vim/bundle/vundle
 $ alias vundle='vim +BundleInstall +qall'
 $ vundle
 ```
